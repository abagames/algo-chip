Interface CompositionOptions

Public composition options resolved by generateComposition.

Defaults when omitted:

  • lengthInMeasures: 32 measures
  • seed: random 32-bit integer per invocation
  • twoAxisStyle: { percussiveMelodic: 0, calmEnergetic: 0 }

Optional sectionRepeatBias

sectionRepeatBias?: number

Controls whether reprised hooks repeat exactly or use varied pitch motifs. 0.0 = Maximum variation, 1.0 = Always repeat exactly. Default: 0.25. Values below 0.25 may choose a varied hook; values 0.25 or above repeat exactly.
